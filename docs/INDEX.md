# Docs Index

This folder contains the core planning and architecture documents for Smart Grocery Assistant V2.

## Directory Structure

| Folder | Purpose |
|---|---|
| `00-specs/` | Product vision, system architecture, AI-layer design, wireframes |
| `01-plans/` | Current phased roadmap and execution plans (Phase 2 onward) |
| `02-notes/` | Dev log, decision records, learnings, open questions |
| `03-evaluations/` | Eval reports, templates |
| `04-walkthroughs/` | Educational walkthroughs of code and concepts |
| `archive/` | Deprecated or superseded documents (Phase 1 plans, historical eval/session exports) |

Within `archive/`:

| Subfolder | Purpose |
|---|---|
| `archive_plan/` | Archived high-level and Phase 1 planning docs |
| `valuable_history/` | Session logs and eval artifacts kept for reference |

## Key Documents

| File | Purpose |
|---|---|
| `00-specs/product-spec-v2.md` | Product spec v3: feature catalog (26 features with IDs), 5 user journeys, acceptance criteria, system behavior |
