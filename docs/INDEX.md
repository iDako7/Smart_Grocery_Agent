# Docs Index

This folder contains the core planning and architecture documents for Smart Grocery Assistant V2.

## Directory Structure

| Folder | Purpose |
|---|---|
| `00-specs/` | Product vision, system architecture, AI-layer design |
| `01-plans/` | Workflow guides and issue templates |
| `02-notes/` | Dev log, decision records, learnings, open questions |
| `03-evaluations/` | Eval reports, templates |
| `04-walkthroughs/` | Educational walkthroughs of code and concepts |
| `archive/` | Deprecated or superseded documents (Phase 1–2 plans, historical artifacts) |

Within `archive/`:

| Subfolder | Purpose |
|---|---|
| `archive_plan/` | Archived high-level, Phase 1, and Phase 2 planning docs |
| `valuable_history/` | Session logs, investigation notes, and eval artifacts kept for reference |

| File | Purpose |
