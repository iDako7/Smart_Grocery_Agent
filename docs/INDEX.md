# Docs Index

This folder contains the core planning and architecture documents for Smart Grocery Assistant V2.

## Directory Structure

| Folder | Purpose |
|---|---|
| `00-specs/` | Product vision, system architecture, AI-layer design, wireframes |
| `01-plans/` | Phased roadmap and detailed execution plans |
| `02-notes/` | Dev log, decision records, learnings, open questions |
| `03-evaluations/` | Eval reports, templates, and session data |
| `04-walkthroughs/` | Educational walkthroughs of code and concepts |
| `archive/` | Deprecated or superseded documents |

## Key Documents

| File | Purpose |
|---|---|
| `00-specs/product-spec-v2.md` | Product vision, target users, core principles, user stories, and feature scope |
| `00-specs/architecture-spec-v2.md` | End-to-end system architecture, components, data flow, interfaces |
| `00-specs/ai-layer-architecture-v2.md` | AI-layer design, orchestration model, tool architecture |
| `01-plans/Implementation_Plan_high_level.md` | High-level phased roadmap (Phase 1–3) |
| `01-plans/phase-1-plan.md` | Detailed Phase 1 execution plan |
